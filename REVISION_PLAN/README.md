## Plan de Révision Recommandé (4 semaines)

### Semaine 1 — Core Java & Spring Boot
- Lundi : Java Core (collections, streams, lambdas)
- Mardi : Java Avancé (concurrence, JVM, GC)
- Mercredi : Spring Boot (IoC, DI, Web MVC)
- Jeudi : Spring Security + Spring Data JPA
- Vendredi : Tests (JUnit 5, Mockito)

### Semaine 2 — Architecture & Patterns
- Lundi : Design Patterns (créationnels, structurels)
- Mardi : Design Patterns (comportementaux) + Clean Code
- Mercredi : Architecture Microservices
- Jeudi : Apache Kafka
- Vendredi : BDD + tests d'intégration

### Semaine 3 — Infra & Outils
- Lundi : SQL/MySQL avancé
- Mardi : Spring Batch
- Mercredi : Docker + Docker Compose
- Jeudi : CI/CD Jenkins + SonarQube
- Vendredi : Git + GitFlow

### Semaine 4 — Révision & Simulations
- Lundi : Angular (notions clés back-end perspective)
- Mardi : Agile/Scrum/Jira
- Mercredi : Simulation entretien — Java + Spring
- Jeudi : Simulation entretien — Architecture + Kafka
- Vendredi : Simulation entretien — Tout le reste
